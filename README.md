# Gestor-de-Turnos-de-Resonancia
Aplicación de consola para la gestión de un servicio de resonancia magnética. Permite registrar pacientes, asignar turnos a los distintos resonadores, consultar los turnos existentes y realizar operaciones de modificación o eliminación.  La lógica del sistema está desarrollada en Python y la persistencia de datos se maneja con MySQL.
